## Five steps

```bash
git clone https://github.com/helderpgoncalves/feedbot.git
cd feedbot
cp .env.example .env

# strong dev secrets
python -c "import secrets; print('FEEDBOT_SECRET_KEY=' + secrets.token_urlsafe(48))"
python -c "import secrets; print('FEEDBOT_BOT_TOKEN='  + secrets.token_urlsafe(32))"

docker compose up --build -d db api
```

Open [http://localhost:8000](http://localhost:8000) → first-run setup creates the owner. From there: invite teammates, create projects, connect Telegram, wire Claude Code.

Full walkthrough in **[Quickstart](/getting-started/quickstart/)**.

## What's *not* free

Operations. Backups, on-call, upgrades, Postgres tuning, observability — that's on you. If you'd rather not, [Feedbot Cloud](/cloud/) handles all of that for a monthly fee. The product itself is identical.
