## Side-by-side

| | Self-host | Cloud |
|---|---|---|
| Price | Free, MIT | Paid, monthly |
| Where it runs | Your infra | Our infra (EU) |
| Postgres | You manage | Managed by us, daily backups |
| Upgrades | You pull, you migrate | Done for you |
| SMTP / email | You configure | Done for you |
| Telegram bot | You bring (one global token) | Shared, or bring your own |
| LLM keys | You bring | You bring (same encryption) |
| SLA | None | Yes, see pricing |
| Data residency | Wherever | EU-first |

## Roadmap

The cloud follows the public roadmap on [GitHub](https://github.com/helderpgoncalves/feedbot#-roadmap):

- **M5** — Multi-tenant managed bot infrastructure.
- **M6** — Billing, Stripe webhooks, plan-based quotas, usage dashboards.

Self-host moves first; cloud follows once the multi-tenant story is proven on real installs.
